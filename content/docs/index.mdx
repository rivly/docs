Rivly is an open-source dashboard for Docker. It manages a single host with a
few compose files and a full Swarm with dozens of nodes the same way, from one
clean, self-hosted interface.

<Callout title="Early development">
  Rivly is being built in the open. These docs grow alongside the product, so
  pages land here as features ship.
</Callout>

## What Rivly does

Rivly gives plain Docker and Docker Swarm the same interface.

- **See everything.** Every container, service, and node in one view, with its
  logs and health.
- **Deploy from Git.** Point Rivly at a repository and it deploys your stacks,
  then keeps them in sync. Your definitions stay in version control, not locked
  in a database.
- **Own your setup.** Self-hosted and MIT licensed. No accounts, no telemetry,
  no lock-in.
